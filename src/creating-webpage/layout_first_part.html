Создание макета страницы для веб–сайта


1. Создание полного макета веб-сайта начинается с создания его начального расположения, затем можно приступать к  проектированию индивидуальных элементов страницы.

2. Ключевые особенности данной веб–страницы – горизонтальные полосы («группы»): они нужны для того, чтобы разместить содержание  странички в определенных областях: красочная область  заголовка - шапка страницы - будет представлять один участок; приветственное обращение с примерами работ – другую часть; и основная часть и ресурс заполнят нижний колонтитул (нижнюю сноску).

3. Работа над любым проектом должна начинаться с эскиза на бумаге. При помощи карандаша можно приблизительно или детально создать план будущего расположения частей веб-странички на бумаге.

4. Планирование каркаса будущей страницы помогает развить иерархию и дает понять, как лучше  расположить ключевые моменты проекта.

5. Создание нового документа. Лучше сделать новый документ размером, соответствующим размеру широкоэкранного монитора, чтобы лучше представить, как будет выглядеть веб–страничка на широком экране. Например, нужно создать новый документ шириной – 1680 пикс.,высотой – 1050 пикс.

6. Воспользовавшись инструментом Guides «Направляющие», нужно разместить эти направляющие линии в центре нового документа. Ширина направляющих составляет приблизительно 960 пикс. С их помощью требуется создать основную сетку для того, чтобы впоследствии было удобнее зрительно расположить основные пункты веб–страницы.

7. Начало работы с создания полосы заголовка: создается новый слой, выбирается инструмент  Rectangular Marquee Tool «Прямоугольное выделение», на новом слое нужно нарисовать выделение, похожее на то, что приведено на рисунке ниже и залить созданное выделение белым цветом. Далее войти в палитру Layer Style «Стили слоя» и добавить стиль слоя Gradient Overlay «Наложение градиента», цвет градиента выбрать от основного к фоновому (от черного к белому) и отрегулировать Opacity «Непрозрачность», чтобы эффект градиента получился не очень темным. После всех манипуляций нужно нажать «Да», чтобы применить эффект Gradient Overlay «Наложение градиента». Градиент рисовать нужно линейного типа.

8. Теперь снова создается новый слой и инструментом Rectangular Marquee Tool  «Прямоугольное выделение» прорисовывается область выделения чуть больше предыдущей: здесь будет располагаться изображаемое содержимое, и в палитре Layer Style«Стили слоя» нужно добавить эффект Gradient Overlay «Наложение градиента». Цвет градиента выбрать двух ярких цветов (на усмотрение); тип градиента: Linear «Линейный». Для того, чтобы добавить немного глубины этому новому слою, в эффектах Layer Style «Стили слоя» требуется войти в графу Inner Shadow «Внутренняя тень» и установить параметры тени.

9.  По желанию: легкая текстура может слегка освежить проект веб–страницы. После того, как к нарисованной прямоугольной области применили  эффект Gradient Overlay «Наложение градиента» из стилей слоя, можно к этому прямоугольнику применить фильтр: Filter> Noise>  Add Noise (Фильтр – Шум – Добавить шум) и режим наложения поставьте Multiply «Умножение» (если после применения данного фильтра «шума» не видно, нужно попробовать отключить видимость эффекта Gradient Overlay «Наложение градиента» в палитре Layer «Слои»).
