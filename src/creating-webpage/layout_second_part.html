Создание макета страницы для веб–сайта


10. Нужно добавить логотип (фирменный знак) компании в той позиции на экране, согласно Grid «Сетке» - «Направляющим», затем добавить к данному фирменному знаку стиль Gradient Overlay «Наложение градиента» и Inner Shadow «Внутреннюю тень» из команды Layer Style «Стили слоя». Внутреннюю тень сделать очень мягкой.

11. Используя инструмент  Type «Текст», создать надпись для главной навигации. Далее нужно установите курсор инструмента Type «Текст» в середине серой полосы с градиентом и написать необходимый текст более темным цветом, чем цвет фона-градиента полоски, на которой сейчас производятся действия.

12. Особенность заголовка заключается в том, что он занимает большее место на веб-странице. Он является основным центром для пользователя. Нужно использовать вторую двуцветную градиентную полоску для того, чтобы разместить на ней вводный заголовок, написанный другим шрифтом.

13. Также нужно к заголовку добавлять детальные пояснения, но, на этот раз, используя шрифты Arial или Helvetica.

14. Следующий пункт разместить необходимую иллюстрацию (в данном случае это будет иллюстрация портативного компьютера).

15. Теперь, для того, чтобы акцентировать внимание на иллюстрации (портативном компьютере), и зрительно выделить ее, нужно создать новый слой, установить его позади слоя с иллюстрацией и, гармонирующим с синим фоном цветом, добавить прямо в центре радиальный градиент. Яркость его и непрозрачность отрегулировать на свое усмотрение.

16. Далее создать новый слой и на нем, под двуцветной центральной частью с иллюстрацией, снова нарисовать небольшую прямоугольную область и заполнить ее серо-белым градиентом.
