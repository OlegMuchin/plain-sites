Оформление горизонтального меню


Начало есть, и теперь можно приступать к вёрстке уже непосредственно основных блоков.

Начать лучше, конечно, с шапки. Которая, в свою очередь, состоит из блоков горизонтального меню и логотипа.

Сначала требуется задать общее оформление обоих элементов шапки: выравнивание текста по левому краю, белый фон и высоту 306px: 

#header {
	background: #ffffff;
	height: 306px;
	text-align: left;		
}

Получено пространство шапки: белый прямоугольник, на котором будут располагаться её элементы.

Будет логично заняться горизонтальным меню, но прежде, чем вставлять его в HTML, нужно подготовить правила CSS, иначе смотреться в браузере это будет плохо. Первые коррективы: задать левую границу в 2 пикселя толщиной, ширину и высоту меню, а также повторяющийся по оси X фоновый рисунок: 

#menu
{
border-left: 2px solid #ffffff;
width: 779px;
height: 80px;
background: url(images/header_top.gif) repeat-x;
}

Страница в обозревателе тут же преобразится и будет выглядеть так.

Теперь можно добавить и само меню в файл index.html:

<table id="menu">
		<tr>
			<td><a href="#"  title="">Главная</a></td>
			<td><a href="#"  title="">Галерея</a></td>
			<td><a href="#"  title="">Договор</a></td>
			<td><a href="#"  title="">Прайс</a></td>
			<td><a href="#"  title="">Образцы</a></td>
			<td><a href="#"  title="">Контакты</a></td>
		</tr>
</table>

Обновив страницу можно увидеть, что оно действительно появилось.

Вид ссылок оставляет желать лучшего. Нужно установить для них свои правила (выравнивание, ширину, цвет, жирность и т. д.), а ссылкам при наведении задать смену цвета и вернуть убранное по всему шаблону подчёркивание: 

#menu a {
	float: left;
	width: 99px;
	height: 46px;
display: block;
	text-align: center;
	text-decoration: none;
	color: #ffffff;
	font-weight: bold;
	font-size: 14px;
	padding-top: 35px;
}

#menu a:hover {
	color: #D72020;
	text-decoration: underline;
}

Теперь форматирование меню можно сопоставить с PSD-шаблоном.

Как меняется оформление пункта, если подвести к нему указатель (за это отвечают правила #menu a:hover).


Настройка логотипа

Логотип уже есть в папке, остаётся добавить его на страницу и отформатировать правилами. И то, и другое можно сделать средствами CSS. 

#logo {
background: #ffffff url(images/big_pic.jpg) no-repeat;
width: 738px;
height: 146px;
text-align: left;
padding-top: 80px;
padding-left: 40px;
border-left: 4px solid #ffffff;
}

Логотип вставлен ровно по размеру.

Единственное, чего ему не хватает, так это текста. Нужно вставить недостающее в блок logo файла index.html, чтобы получилось: 

<div id="logo">
		<h1><a href="#">Имя сайта</a></h1>
		<h2><a href="№" id="metamorph">Слоган сайта</a></h2>
</div>

Текст появился, но его тоже нужно оформлять.

Исходя из того, что логотип почти всегда бывает ссылкой, требуется задать оформление соответствующим классам. 

#logo a {
	text-decoration: none;
	text-transform: lowercase;
	font-style: italic;
	font-size: 36px;
	color: #FFFFFF;
}
#logo h2 a
{
font-size: 24px;
}

Внешний вид текста изменился, и в целом шапка теперь выглядит даже лучше, чем на PSD-макете.
