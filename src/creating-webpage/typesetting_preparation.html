Верстка дизайна


После создания макета сайта в программе Photoshop надо сверстать его из PSD-макета в готовую веб-страницу, используя инструменты HTML-разметки и каскадные таблицы стилей (CSS)

Подходов к подготовке шаблонов существует множество:

  • сначала досконально прорисовывается шаблон и только потом вёрстка;
  • отрисовка дизайна приблизительно, добавляя штрихи во время вёрстки;
  • сначала рисование, затем нарезка;
  • сохранение графических объектов непосредственно в процессе создания дизайна.

Инструменты вёрстки и рисования у каждого свои:

  • установка плагинов и работа в визуальных редакторах;
  • использование стандартного набора возможностей и верстка в простом редакторе.

Нужно создать где угодно (хоть на Рабочем столе, хоть в отдельной папке) файл index.html. В этот же каталог добавить директорию images. Она будет содержать все картинки, используемые в шаблоне и на странице. Так как графические элементы вырезали заблаговременно, их нужно скопировать в папку images с названиями:

  • back_all — подложка сайта;
  • header_top — фон шапки;
  • big_pic — логотип;
  • title — фон заголовков левой панели;
  • footer — заливка низа сайта;
  • 1mini — первое фото для основной части страницы;
  • 2mini — второе фото.

В папке со страницей index.html нужно создать файл styles.css — в нём будут размещены таблицы стилей шаблона.

В папке со страницей index.html нужно создать файл styles.css — в нём будут размещены таблицы стилей шаблона.

Для редактирования кода лучше подходит редактор Notepad++. Это подсвечивающая синтаксис разных языков (HTML и CSS — в том числе) программа.

Разделение документа на блоки

Слудующим этапом требуется открыть документ index.html и вставить в него следующий код: 

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Шаблон сайта</title>
<meta name="keywords" content="">
<meta name="description" content="">
<link href="styles.css" rel="stylesheet" type="text/css" media="screen">
</head>
<body>
</body>
</html>

Следующим этапом нужно сохранить файл.

После создания основы HTML-страницы, теперь нужно разделить её на блоки — указать структуру документа, что за чем в нём будет идти.

Блоков 7, перечисление их по идентификатору (значению атрибута id):

  1. content — блок, внутри которого будут храниться остальные блоки;
  2. header — блок шапки, внутри которого будут;
  3. menu — верхняя навигация;
  4. logo — картинка с текстом;
  5. right — основная часть страницы;
  6. left — панель слева;
  7. footer — низ сайта.

Теперь в контейнер <body> нужно вставить код:

<div id="content">
<!-- Шапка -->
	<div id="header">
	  <div id="menu">
	  </div>
	  <div id="logo">
	  </div>
	</div>
<!-- Конец шапки -->
<!-- Основной блок -->
	<div id="right">
	</div>
<!-- Конец основного блока -->
<!-- Левая панель -->
	<div id="left">
	<div>
<!-- Конец левой панели -->
<!-- Ноги сайта -->
	<div id="footer">
	</div>
<!-- Конец -->
</div>

В браузере страница будет по-прежнему пустой, но структуру документа уже можно понять, она готова.

Установка базового форматирования

Следующий этап CSS-оформление, оно нужно чтобы задать документу начальное оформление. Нужно открыть style.css и добавить туда строки кода.

Удаление отступа и полей на странице по умолчанию:

*
{
margin: 0px;
padding: 0px;
}

Установка цвета ссылок в зависимости от поведения пользователя (навёл курсор, не навёл, посетил) и удаление подчёркивания у ссылок, над которыми находится указатель мыши: 

a:link {
	color: #D72020;
}

a:hover {
	text-decoration: none;
	color: #FF0000;
}

a:visited {
	color: #D72020;
}

Настройка основного оформления страницы: указание фонового цвет и изображения-подложки (картинке задать горизонтальное заполнение), устанавливаем цвет, стиль и размер шрифта: 

body {
	background: #FFD723 url(images/back_all.jpg) repeat-x;
	font: 13px Tahoma, Verdana, Arial, Helvetica, sans-serif;
	color: #333333;
}

Определение блока content:

#content {
	margin: 0 auto;
	background: #ffffff;
	width: 786px;
	text-align: left;
}

Вот теперь можно обновить страницу. Она заполнена рисунком-подложкой. Пока единственное видимое изменение, за которое отвечает свойство background класса body.
